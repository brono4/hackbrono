1. Maintain Access

After gaining access, it's essential to create a backdoor or persistent method to ensure you can return later.

  • Create a backdoor: Upload a malicious web shell or SSH key for persistent access.
  • Scheduled tasks: Use scheduled tasks to re-launch your payload.
  • Planting rootkits: Install rootkits to avoid detection and maintain long-term access.

2. Extract Sensitive Information

Once you have access, gather information such as credentials, personal data, or proprietary information.

  • Dump password hashes: Use tools like mimikatz or hashdump to extract password hashes.
  • Capture network traffic: Sniff the network for sensitive data like plaintext credentials using tcpdump or Wireshark.
  • Database extraction: Query databases for sensitive information using SQL injection or shell access.
