Popular OSINT Tools

Nmap

Nmap ("Network Mapper") is a free and open-source tool used for network discovery and security auditing. It can identify hosts and services on a network, enabling the mapping of the network and the detection of vulnerabilities.

Recon-ng

Recon-ng is a full-featured web reconnaissance framework designed to gather information through modules. It automates many common tasks, helping penetration testers collect data during the reconnaissance phase.

Maltego

Maltego is an open-source intelligence (OSINT) and graphical link analysis tool. It helps users visualize relationships between data points, making it easier to investigate networks, domains, IP addresses, and more.

Shodan

Shodan is a search engine that scans the internet for connected devices. It allows users to find information on exposed systems, such as webcams, routers, or even power plants, by searching the internet for open ports and other network data.
