Scanning Commands and Tools

Network Scanning: 

nmap -sV -sC -O -T4 -oA scan  # Fast scan for the most 1000 TCP ports used
nmap -sV -sC -O -T4 -p - -oA fullfastscan  # Fast scan for all the ports
nmap -sV -sC -O -p - -oA fullscan  # Scan every TCP port to know exactly what is running in that machine
nmap -sV -sC -T # Check if any of the 1000 most common UDP ports are running
